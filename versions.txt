# Orchid Mantis / zkpox toolchain manifest

# Pinned to match what produces reproducible artifacts. The SP1 pins
# are load-bearing: the verifier ships a hard-coded mapping from
# backend kind/version to the SP1 verifying-key digest that backend
# produces, and any drift in the SP1 release would break that
# mapping.

# Rust host toolchain (matches workspace rust-toolchain.toml).

# SP1 (zkVM).
sp1-zkvm:             6.0.1

# Cross-compile toolchain for the freestanding C target.

                      Apple's bundled clang lacks the RISC-V backend; the
                      build.rs picks Homebrew clang explicitly on macOS
                      and falls back to PATH `clang` elsewhere.
                      Override via ZKPOX_CLANG env var.
